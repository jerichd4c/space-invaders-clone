# 👾 Space Invaders Clone - Web Development 1 Project

This project is a high-fidelity clone of the classic arcade game *Space Invaders*, developed from scratch using vanilla web technologies without any additional frameworks.

## 🛠️ Technologies Used
* **HTML5 Canvas:** Core game rendering and structure.
* **CSS3:** Retro-themed interface, menus, and typography.
* **JavaScript (ES6+):** Game logic, OOP (Object-Oriented Programming), physics, collision detection, and custom game loop.
* **Local Storage:** High score persistence.

## ✨ Key Features
* **Fluid Mechanics:** Responsive player movement and firing system.
* **Progressive Difficulty:** Alien horde with multiple levels, faster movement, and increased firing rates as you progress.
* **Pixel-Art Sprites:** Custom sprite animations for enemies (idle/movement), player (active/death), and explosion effects.
* **Destructible Barriers:** Protective barriers that degrade dynamically when hit by either player or enemy projectiles.
* **Classic Soundscape:** Retro sound effects for shooting, explosions, crashes, and alien movement.
* **Full Game Cycle:** Main menu, high score tracking, gameplay stages, and Game Over states.

## 🎮 Controls
* **Left/Right Arrow Keys:** Move the player ship.
* **Spacebar:** Fire projectile.
* **Enter:** Navigate and select options in the menu.

## 🚀 How to Run the Project Locally
1. Clone this repository to your local machine:
   `git clone <repository-url>`
2. Open the project folder in your code editor (e.g., VS Code).
3. Open the `index.html` file directly in your web browser, or use an extension like *Live Server* for automatic hot-reloading.
